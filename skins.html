Использовать оружие просто: для стрельбы нужно нажать ЛКМ, для альтернативного режима стрельбы - ПКМ, а для переключения между оружием используется колёсико мыши. Эти кнопки могут быть изменены в настройках игры. Так же оружие распределяется по слотам: слот для основного оружия (кнопка 1), вспомогательное (кнопка 2), ближний бой (кнопка 3), гранаты (кнопка 4), бомба (кнопка 5). Игрок может нести только одно оружие одного типа (кроме гранат). К оружию ближнего боя относится нож, который есть всегда. Вспомогательным оружием являются пистолеты, а основное делится на дробовики, пистолеты-пулемёты, штурмовые и снайперские винтовки, пулемёты. Если какое-либо оружие не нужно, его можно выбросить кнопкой «G». Для подбора любого оружия с земли, у вас должен быть свободный слот.
